# Scoops & Smiles 🍦

A modern and responsive artisanal ice cream parlor website built using HTML, CSS, and JavaScript.

https://scoops-and-smiles.vercel.app/

* Mobile Navigation Menu

* Interactive Menu Filtering
* Modern UI Design
* Best Seller Section
* Contact & Footer Section
* Clean Layout
